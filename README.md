作为搭建JavaScript第三方库的脚手架。

## 功能

- 支持ES6及以上版本语法
- 支持`Tree Shaking`
- 支持输出多种模块（IIFE、AMD、CMD、UMD、ESM）规范格式，development（.js）与production（.min.js）版本
- 支持自定义`banner`
- 支持`eslint`
- 支持可持续构建`travis-ci`
- 提供`issue-template`
- 支持单元测试

## 快速使用

```shell
npm install @careteen/jslib-cli -S
```

```js
import { test } from '@careteen/jslib-cli'
```
